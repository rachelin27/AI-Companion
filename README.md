# 星愈 - 智能心理服务助手
一款基于Streamlit和DeepSeek大模型开发的智能心理服务对话系统，提供个性化、多角色的心理陪伴与疏导服务，支持会话管理、角色定制、数据本地存储等功能。

##  项目简介
星愈（Xingyu）是一款轻量级的智能心理服务工具，旨在为用户提供温暖、专业的心理陪伴。系统支持自定义AI角色性格、会话自动保存/管理、上下文记忆、流式回复等核心功能，所有对话数据本地存储，保障隐私安全。

##  项目开发与更新过程
本项目的开发迭代拆分为四个核心阶段，逐步完善功能与体验：
1. **页面基础搭建与交互实现**：完成页面基础配置（标题、图标、布局等），实现核心的聊天输入、AI流式回复等基础交互功能，打通与DeepSeek大模型的调用链路，保障基础对话能力可用；
2. **侧边栏功能拓展**：新增侧边栏模块，集成预设角色切换、自定义身份/性格设置、对话参数调整（上下文长度、创造性等）等功能，提升系统的个性化配置能力；
3. **会话功能升级与历史记录实现**：开发会话管理核心能力，包括会话的新建、加载、删除、标题自动生成/手动编辑，以及会话数据的本地JSON存储，实现历史对话记录的持久化与可视化；
4. **系统整体升级优化**：补充批量会话管理、会话导出、对话统计、自动保存等进阶功能，优化交互逻辑与异常处理，完善隐私保护机制，提升系统的稳定性、易用性与安全性。
